Reference summary distilled from the linked Apple design preview:

## Visual Direction

- Premium white space
- SF Pro style system typography
- White and light-gray surfaces
- Blue primary CTA accent
- Frosted or blurred navigation and overlay surfaces

## Core Tokens

- Primary text: `#1D1D1F`
- Background white: `#FFFFFF`
- Background gray: `#F5F5F7`
- Accent blue: `#0071E3`
- Border gray: `#D2D2D7`
- Frosted overlay: `rgba(210, 210, 215, 0.64)`
- Navigation blur: `blur(20px)`

## Layout Guidance

- Prefer full-screen sections with large image focus
- Use generous vertical spacing
- Keep controls in clean rounded groups, not dense dashboards
- Separate primary actions from secondary utilities

## Component Guidance

- Primary buttons: pill radius, blue fill, white text
- Secondary buttons: pill radius, blue outline or soft gray fill
- Cards: soft corners, subtle shadows, light backgrounds
- Toolbars: translucent white glass, thin border, blur
- Text hierarchy: large headline, restrained secondary copy

## App Adaptation

- Default to a light appearance
- Use glass cards for image stage, queue, and tool groups
- Keep the bottom toolbar compact so content can remain scrollable
- Avoid dark gradient hero treatments for this version
